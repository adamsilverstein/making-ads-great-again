How ads work

A typical page load with ads:

Page requested by the browser

Scripts will be in header, slots and scripts on the page

Header has gpt ad script injection

Header has inital (batch) ad request

Page body has several ad slots - divs with ad code

DFP deh4vers an ad for each slot: either from inventory based on targeting, or from the network ('remnant ads')
